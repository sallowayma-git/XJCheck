# 线网抽取算法规格

## 1. 输入

- 原始 CAD primitives；
- page audit zone；
- layer metadata；
- block instances；
- symbol library（可为空）；
- project profile；
- tolerance profile。

## 2. 输出

- atomic geometry nodes/edges；
- junction observations；
- topology decisions；
- asserted electrical networks；
- possible/rejected boundaries；
- open endpoints；
- witness paths。

## 3. 算法流水线

### Step A：归一化

1. LINE 直接转 segment；
2. LWPOLYLINE/POLYLINE 拆分为 segment，保留 parent；
3. ARC/CIRCLE 保留曲线或按容差离散；
4. nested INSERT 展开变换，但同时保存 symbol ownership；
5. 去除零长度/无效实体；
6. 按 layer policy 标记 `wire_candidate / symbol_graphic / frame / table / unknown`，不要立即删除 unknown。

### Step B：空间候选

- endpoint index；
- segment bbox index；
- block bbox/port index；
- dot/arc index。

生成候选时使用 adaptive tolerance：

```text
tolerance = max(project_absolute_min,
                median_text_height * factor,
                median_line_gap * factor)
```

### Step C：交点与切分

1. 求 endpoint-endpoint snap；
2. 求 endpoint-on-segment；
3. 求 segment intersection；
4. 分类连接/穿越；
5. ASSERTED intersection 变为节点；
6. 在节点处切分长线为 atomic edge。

### Step D：符号端口

- 匹配 block family；
- 变换 local ports；
- 找进入端口的线；
- 根据 internal connectivity 生成确定或条件连接；
- unknown symbol 保持 boundary。

### Step E：Gap 候选

只生成 Possible edge。特征包括：

- gap length；
- collinearity；
- layer；
- intervening symbol/text；
- endpoint degree；
- parallel competitors；
- repeated family pattern；
- page zone。

### Step F：TopologyDecision

决策优先级：

```text
explicit symbol/port rule
> explicit dot/jumper rule
> exact endpoint/T geometry
> project-verified structural rule
> calibrated model
> proximity heuristic
```

低优先级不得覆盖高优先级。

### Step G：Net materialization

只把 ASSERTED edges 放入 connected components。POSSIBLE edges 作为 adjacent candidates 记录。

## 4. 关键反例

### 反例 1：十字穿越

线条几何相交不代表连接。无 dot 且两条线连续通过，应默认不合并或 unknown。

### 反例 2：文字造成断线

文字位于 gap 中不能自动证明导线连续。只有 symbol/重复模式/约束支持时才能 bridge。

### 反例 3：块 insertion point

块插入点不等于端口。必须使用 family local port 或外部入线聚类推断。

### 反例 4：表格

大量水平/竖直线不等于电气网。页面结构层必须标识 table/grid，表格线不能进入 wire network。

### 反例 5：通信图

连线可能表示逻辑、光纤或以太网。必须有 medium，不能与普通 conductor 统一合并。

## 5. 调试产物

每页应输出：

- `primitive_overlay.svg`；
- `junction_overlay.svg`；
- `network_overlay.svg`；
- `topology_decisions.json`；
- `network_stats.json`；
- `unresolved_topology.csv`。
